Introduction

Git et son fonctionnement :

Git est utilisé pour versionner son code.

Commande de GIT via le terminal

git clone URL Récupère un dépôt à partir d'une adresse URL
git add fichier Marque le fichier à consigner
git add "nom_du_ficher" Marque tous les fichiers à consigner
git commit -m "­Commentaire" Consigne les fichiers marqués dans le dépôt
git status Affiche l'état du dépôt local
git tag étiqu­ette Pose une étiqu­ette de version sur les fichiers
git push origin master Envoie les modifi­cations sur le dépôt principal
